    JDBC is a java API that is used to connect or intersect with the DataBases. (JDBC DRIVERS KA USE KRKE CONNECT KR SAKTE HAI)
    JDBC Drivers are software components that provides the necessary functonality to connect with the java applications.

    TYPES OF DRIVERS :

        i . JDBC ODBC bridge driver : 

            - Ye ek Old / Outdated driver hai 
            - c programming se banaya gya hai 
            - performence wise bekar hai
            - scaleablity issues hai. 

        ii. Native API : 
        
            - Jis database ka API use kr rhe hai wahi data base use krna padta hai (mysql ke api ko mysql database, postgreySQL, Oracle ,etc)
            - Vender specific hai. 

        iii. Native Protocol Pure Java Driver : 

            - Better > JDBC ODBC AND NATIVE API 
            - java language se banaya hua hai.
            - ye Client aur Database ke beech mai Middleware (Layer) banata hai aur dono ko connect krta hai. 

        iv. Thin Driver : 

            - Ye Directly Native API calls ko convert karta hai Database calls ke andar 
            - No Middleware
            - No scaleablity issues 
            - Better than all of above Drivers
